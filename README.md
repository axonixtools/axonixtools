**Directory:** [axonixtools.com/agent-skills](https://axonixtools.com/agent-skills)
**Detail pages:** [axonixtools.com/agent-skills/[slug]](https://axonixtools.com/agent-skills)
 
### What Makes Skills Different
 
Unlike simple prompt suggestions, Axonix Agent Skills provide:
 
*   **Complete rules files** — copy/paste-ready instructions for your assistant.
*   **Trigger phrases** — know exactly when to activate each skill.
*   **Compatible assistants** — works with Claude, Cursor, Windsurf, GitHub Copilot, Cline, Aider, Continue, and more.
*   **Configuration file targets** — AGENTS.md, .cursorrules, CLAUDE.md, .windsurfrules, and others.
*   **Difficulty ratings** — beginner, intermediate, or advanced.
*   **Real use cases** — practical scenarios, not vague advice.
